# Projet JavaScript - JustOne

## Description du projet
Le but de ce projet est de découvrir les concepts clés de JavaScript. Pour cela on a codé un min-jeu, le JustOne.

## Lancer le projet
Pour lancer le projet :
- `npm install`
- `npm start`

## Améliorations possibles
- implémenter le jeu sous la forme d'un serveur, pour jouer sur plusieurs machines
- trouver une base de données plus appropriée (sans les verbes conjugués)
- gestion de l'écriture des lettres accentuées dans le terminal
- écriture d'un fichier "historique" plus détaillé
